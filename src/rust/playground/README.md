# Rust

## Installation

```bash
 curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
 xcode-select --install
```

## Creating a project

```bash
cargo new ...
```

## Run App

### Compile program

```bash
cargo run
or (if not using cargo, this will also execute the app)
rustc main.rs
```

### Execute program
```bash
cargo run
or
./main
```

### Make sure your program will compile
```bash
cargo check
```
