
# Polyglot Frontend
## Notebook UI:
     - displays and edits cells, each cell can be code in a given language or rich markdown
     - show outputs/errors
     - lets you select a language per cell
### Notebook tools per language
    - Python: Jypyter notebooks
    - Lua:
    - Go: GoNB
    - JS/TS: Quarto, Starboard, Observable notebooks
    - Rust:
    - Zig

      "type": "code",
